# Notex

### Installation

Install Python 3.6 or higher

```bash
$ sudo apt-get install python3.6
```

Install pip

```bash
$ sudo apt-get install python3-pip
```

Install the requirements

```bash
$ pip install -r requirements.txt
```


### Running the app

Running the Web APP

```bash
$ ./run.sh
```

### Running only the UI

Running the Local UI

```bash
$ ./run_ui.sh
```
